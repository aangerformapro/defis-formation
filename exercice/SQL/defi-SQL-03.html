Exercice 02

Modifier l'étudiant avec l'identifiant 5, en changeant son nom par Merucci et sa date de naissance par 07-03-1985 

        UPDATE `acs_etudiants SET nom = 'Merucci', date_de_naissance='1985-03-07' WHERE id_etudiant = 5
