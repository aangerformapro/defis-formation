Défi SQL - 01

Ci-après la table à utiliser pour la première partie des exercices.

Table : acs_livres
id_livre titre isbn_10 auteur prix
1 Forteresse digitale 2709626306 Dan Brown 20.5
2 La jeune fille et la nuit 2253237620 Guillaume Musso 21.9
3 T'choupi se brosse les dents 2092589547 Thierry Courtin 5.7
4 La Dernière Chasse 2226439412 Jean-Christophe Grangé 22.9
5 Le Signal 2226319484 Maxime Chattam 23.9

Exercice 01

Quelle requête utiliser pour afficher l'ensemble des enregistrements de la table acs_livres ?

Réponse

SELECT * FROM acs_livres

Exercice 02

Quelle requête utiliser pour sélectionner uniquement les livres qui ont un prix strictement supérieur à 20 de la table acs_livres ?

Réponse

SELECT * FROM acs_livres WHERE prix > 20

Exercice 03

Quelle requête utiliser pour trier les enregistrements de la table acs_livres du prix le plus élevé aux prix le plus bas ?

Réponse

SELECT * FROM acs_livres ORDER BY prix DESC

Exercice 04

Quelle requête utiliser pour récupérer le prix du livre le plus élevé de la table acs_livres ?

Réponse

SELECT prix FROM acs_livres ORDER BY prix DESC LIMIT 1

Exercice 05

Quelle requête utiliser pour récupérer les livres de la table acs_livres qui ont un prix compris entre 20 et 22 ?

Réponse

SELECT * FROM acs_livres WHERE prix >= 20 AND prix <= 22

Exercice 06

Quelle requête utiliser pour récupérer tous les livres de la table acs_livres à l'exception de celui portant la valeur pour la colonne isbn_10 : 2092589547 ?

Réponse

SELECT * FROM acs_livres WHERE isbn_10 != 2092589547

Exercice 07

Quelle requête utiliser pour récupérer le prix du livre le moins élevé de la table acs_livres en renommant la colonne dans les résultats par minus ?

Réponse

SELECT prix as minus FROM acs_livres ORDER BY prix ASC LIMIT 1

Exercice 08

Quelle requête utiliser pour sélectionner uniquement les 3 premiers résultats sans le tout premier de la table acs_livres ?

Réponse

SELECT * FROM acs_livres WHERE id_livre > 1 ORDER BY id_livre ASC LIMIT 3
